  # 🧭 Outdoor Comp - Archivos de Expedición

Bienvenido al repositorio oficial de **Outdoor Comp**, tu base de suministros para el siglo XXI con el espíritu de los grandes exploradores de 1930.

Este proyecto es una tienda virtual funcional diseñada para aventureros, montañistas y amantes del rivering.

## 🚀 Funcionalidades Actuales
- **Portada Animada:** Una bienvenida inmersiva al estilo "Indiana Jones".
- **Catálogo Dinámico:** Filtrado de productos por categoría (Camping, Senderismo, etc.).
- **Sistema de Carrito:** Gestión de suministros en tiempo real.
- **Integración con WhatsApp:** Finalización de pedidos directa al cuartel general.
- **Asesor Virtual:** Chatbot interactivo (Jones) para asistencia en campo.
- **Mapa de Expedición:** Ubicación estratégica de nuestra base de operaciones.

## 🛠️ Tecnologías Utilizadas
- **HTML5 & CSS3:** Estructura y diseño con texturas vintage y tipografías desgastadas.
- **JavaScript (Vanilla):** Lógica del carrito, filtros y chatbot.
- **GitHub Pages:** Hosting gratuito y despliegue automático.

## 📂 Estructura del Proyecto
- `index.html`: El núcleo de la estación de mando.
- `*.jpg / *.png`: Archivos visuales de las reliquias y suministros.
- `README.md`: Este manual de instrucciones.

## 📝 Notas de la Misión
Este sitio fue construido como parte de un proyecto de aprendizaje para dominar herramientas de desarrollo web y despliegue en la nube.
